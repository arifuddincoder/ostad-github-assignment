# Ostad GitHub Assignment

This repository is created as part of an assignment on **Version Control using Git and GitHub**. It demonstrates the core Git workflow: committing changes, creating branches, merging, and working with pull requests.

## About

The goal of this project is to practice and showcase essential Git and GitHub skills, including:

- Initializing a local Git repository
- Staging and committing files with meaningful messages
- Linking a local repository to a remote GitHub repository
- Creating and switching between branches
- Pushing branches to GitHub
- Opening and merging pull requests

## Project Structure

- `README.md` — Project description and documentation
- `hello.txt` — A sample file added on the `dev` branch

## Workflow Followed

1. Created a local repository and committed `README.md`
2. Linked the local repo to a remote GitHub repository
3. Created a `dev` branch and added `hello.txt`
4. Pushed the `dev` branch to GitHub
5. Opened a pull request to merge `dev` into `main`
6. Merged the pull request and updated the local `main` branch
